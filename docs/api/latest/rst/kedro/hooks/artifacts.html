Bases: LoadArtifactSpec, EmptyStrAsMissing

Specification of an artifact to be fetched from MLFlow and placed before a pipeline runs.

See also

  • mlopus.mlflow.BaseMlflowApi.place_run_artifact()

  • mlopus.mlflow.BaseMlflowApi.place_model_artifact()

  • If schema_ is specified, it is used to verify the artifact after placing it.

  • subject and skip_reqs_check are used if schema_ is an alias.
