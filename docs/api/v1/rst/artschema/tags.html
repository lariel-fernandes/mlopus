Tags

pydantic model mlopus.artschema.Tags[source]

Bases: BaseModel, MappingMixin

Artifact schema tags for MLflow experiments, runs, registered models and model versions.

Example:

# Get a model version API.
version = mlopus.mlflow \
    .get_api(...) \
    .get_model(...) \
    .get_version(...)

# Register a schema for that version.
mlopus.artschema.Tags() \
    .using(MySchema, aliased_as="my_schema") \
    .register(version)

# Inspect the resulting tags.
version.tags["schemas"]
{
    "my_schema": {
        "cls": "my_schemas:MySchema",
        "pkg": {  # this package requirement will be validated when importing the schema class
            "name": "my_schemas",
            "version": "1.2.3",
            "constraint": ">=",
            "extras": [],
        }
    }
}

# Load the model using the aliased schema
mlopus.artschema.load_artifact(version, schema="my_schema")

# Package requirements can be ignored with `skip_reqs_check=True`
# Package details are inferred automatically unless specified when calling `using(...)`.

using(cls, aliased_as=None, from_package=None, at_version=None, with_constraint='>=', and_extras=None)[source]

Add aliased artifact schema to this Tags object.

If not specified, the params from_package and at_version are inferred from the metadata of the provided cls and the packages installed in the current environment.

Parameters:

  • cls¶ (Type[Schema]) – Artifact schema class.

  • aliased_as¶ (Optional[str]) – Schema alias. Defaults to default.

  • from_package¶ (Optional[str]) – Required package for this schema.

  • at_version¶ (Optional[str]) – Required package version.

  • with_constraint¶ (Literal['==', '~=', '^', '>=']) – Version requirement constraint.

  • and_extras¶ (Optional[Sequence[str]]) – Required package extras for using the schema.
